# メール要約・優先順位ツール 操作マニュアル

AIがGmailの受信メールを自動で要約・優先度分類するツールです。
複数のGmailアカウントを一括管理し、重要なメールを素早く見つけることができます。

## 1. 初回セットアップ

### 必要なもの

- Googleアカウント（Gmail）
- ブラウザ（Chrome推奨）

### 起動方法

ツールを開くと、ログイン画面が表示されます。

---

## 2. ログイン

### 手順

1. 「**Googleアカウントでログイン**」ボタンをクリック
2. Googleのアカウント選択画面が表示される
3. ログインに使うGmailアカウントを選択
4. アクセス許可画面で「**許可**」をクリック
5. ダッシュボードに自動で遷移

### 初回ログイン時の注意点

**初回はエラーが出ることがあります。** これは正常な動作です。

- Googleの認証トークンの取得に時間がかかる場合がある
- **対処法：** ページを再読み込み（F5 または Cmd+R）してください
- 再読み込みで正常にダッシュボードが表示されます

### アクセス許可について

このツールは以下の権限を使用します：

| 権限 | 用途 |
|-----|------|
| メールの閲覧 (gmail.readonly) | メール内容の取得・要約に使用 |
| メールの変更 (gmail.modify) | 未読→既読の変更に使用 |
| プロフィール情報 | ログインユーザーの識別に使用 |

※ メールの送信・削除は行いません。

## 3. ダッシュボードの見方

### ヘッダー

### アカウント選択エリア

- メールを取得する期間を指定するエリアです

## 4. メールを分析する

### 期間の指定方法（3種類）

タブを切り替えて期間を選びます。

#### (A) 直近○時間

最もよく使う方法です。

- 「1時間」「3時間」「6時間」「12時間」「24時間」「48時間」「72時間」「1週間」から選択
- **おすすめ：** 朝一で「12時間」、こまめに確認なら「3時間」

#### (B) ○日○時〜今

特定の日時以降のメールを取得します。

- 日付と時刻を入力
- 例：「3月29日 09:00」以降のメール

#### (C) 期間指定

開始日〜終了日の範囲でメールを取得します。

- From（開始日）とTo（終了日）を入力

### 分析の実行

1. 期間を選択
2. 「**分析を開始する**」ボタンをクリック
3. AIがメールを取得→要約→優先度分類を自動実行
4. 処理が完了するとメール一覧が表示される

※ メール件数が多い場合、30秒〜1分ほどかかることがあります。

### 優先度の分類

AIが各メールを4段階の優先度に自動分類します。

| 優先度 | バッジ | 意味 | 例 |
|-------|-------|------|-----|
| 🔴 即対応 | urgent | すぐに返信・対応が必要 | クライアントからの急ぎの依頼、締切間近の案件 |
| 🟡 今日中 | today | 今日中に確認・対応すればOK | 会議の日程調整、通常の業務連絡 |
| 🔵 確認のみ | info | 読むだけでOK、返信不要 | ニュースレター、通知メール |
| ⚪ スキップ | skip | 読まなくてOK | 広告、自動送信メール |

## 5. メール一覧の操作

### フィルタリング

分析結果の上部にフィルターボタンがあります。

- **全N通**：全メールを表示
- **🔴即対応N通**：urgentのみ
- **🟡今日中N通**：todayのみ
- **🔵確認のみN通**：infoのみ
- **⚪スキップN通**：skipのみ

### タブ切り替え

- **全て**：全メール
- **未読**：未読メールのみ
- **既読**：既読メールのみ

### メール一覧の表示内容

各メールカードには以下が表示されます：

- **優先度バッジ**（🔴🟡🔵⚪）
- **送信者名**
- **アカウント表示**（複数アカウント時、どのGmailに届いたか）
- **受信時刻**
- **件名**
- **AI要約**（メール内容の1〜2行の要約）
- **未読マーク**（青い丸印）

### 対応済みにする

各メールの「対応済み」ボタンをクリックすると、そのメールに取り消し線が入り、対応済みとして管理できます。

### 一括既読にする

未読メールをまとめて既読にできます。Gmail側にも反映されます。

## 6. メールの詳細を見る

### プレビュー表示

メール一覧からメールをクリックすると、右側にプレビューパネルが開きます。

### プレビューの内容

- **件名**
- **From**（送信者）
- **To**（宛先）
- **Cc**（あれば）
- **Date**（送信日時）
- **添付ファイル**（📎 アイコンで表示）
- **本文**（HTML形式またはテキスト形式）

### Gmailで開く

プレビュー右上の「**Gmailで開く**」ボタンをクリックすると、ブラウザの新しいタブでGmailの該当メールが開きます。返信が必要な場合はここから直接返信できます。

### プレビューを閉じる

「✕」ボタンをクリックするか、別のメールをクリックすると切り替わります。

## 7. 複数アカウントの管理

このツールの最大の特徴は、**複数のGmailアカウントのメールを一括で確認できる**ことです。

### アカウントの追加

1. ダッシュボードのアカウント選択エリアをクリック
2. 「**別のGmailアカウントを追加**」をクリック
3. Googleのアカウント選択画面が表示される
4. 追加したいGmailアカウントを選択して許可
5. ダッシュボードに戻り、アカウントが追加されている

※ 追加できるアカウント数に制限はありません。

### アカウントの切り替え

1. アカウント選択エリアをクリック
2. 表示したいアカウントにチェックを入れる
3. チェックを外すとそのアカウントのメールは非表示になる

### 複数アカウントを同時に表示

- 複数のアカウントにチェックを入れると、**全アカウントのメールがまとめて表示**されます
- 各メールにはどのアカウントに届いたかがバッジで表示されるので、混ざっても区別できます
- 「**全選択**」で全アカウントのメールを一括表示

### アカウントの削除

1. アカウント選択エリアをクリック
2. 削除したいアカウントの「✕」ボタンをクリック
3. 確認後、アカウントが削除される

※ 削除してもGmailのメール自体は消えません。このツールからの連携が解除されるだけです。

### アカウント追加時のエラーについて

**初めてアカウントを追加する際、エラーが表示されることがあります。**

- Googleの認証情報の保存に少し時間がかかるためです
- **対処法：** ページを再読み込みしてください。正常に追加されています

## 8. よくあるエラーと対処法

### 「トークンの更新に失敗しました」

**原因：** Googleの認証が期限切れになった
**対処法：**
1. 一度ログアウト
2. 再度ログイン
3. アカウントを再追加

### 「メールの取得に失敗しました」

**原因：** Gmail APIの一時的なエラー、またはネットワーク問題
**対処法：**
1. ページを再読み込み
2. 数分待ってから再度「分析を開始する」をクリック

### 分析結果が表示されない

**原因：** 指定した期間にメールが存在しない
**対処法：** 期間を広げて再分析（例：「3時間」→「24時間」）

### AIの要約が件名と同じになる

**原因：** AI分析が一時的にスキップされた（API負荷時）
**対処法：**
- 自動でフォールバック処理が行われるため、要約の代わりに件名が表示されます
- 時間をおいて再分析すると正常に要約されます

### 複数アカウント時に一部のメールが表示されない

**原因：** 一部アカウントのトークンが期限切れ
**対処法：**
1. 該当アカウントを一度削除
2. 再度「別のGmailアカウントを追加」から追加

### 初回ログイン/アカウント追加でエラーが出る

**これは既知の挙動です。初回のみ発生します。**
**対処法：** ページを再読み込み（F5 または Cmd+R）するだけでOK

## 9. Tips・活用法

### 朝のメール確認ルーティン

1. ツールを開く
2. 「直近12時間」で分析を開始
3. 🔴即対応のメールから処理
4. 🟡今日中のメールを確認
5. 🔵確認のみ・⚪スキップは後回し

→ これだけで朝のメールチェックが **5分で完了** します。

### 複数アカウント活用法

- **仕事用 + プライベート用** を同時表示 → 朝一で全メールを一括確認
- **本アカウント + サブアカウント** → 全てのメールを見落とさない
- **個人 + チーム共有** → 複数の受信トレイを1画面で管理

### 期間指定の使い分け

| シーン | おすすめ設定 |
|-------|------------|
| 朝一の確認 | 直近12〜24時間 |
| 会議前のチェック | 直近3時間 |
| 週末明けの確認 | 直近72時間 or 1週間 |
| 特定日のメール探し | 期間指定 |

---

## 補足：セキュリティについて

- このツールはメールの**閲覧と既読変更のみ**を行います
- メールの送信・削除・転送は一切行いません
- Googleの公式OAuth2認証を使用しています
- ログアウトすると認証情報は無効化されます
- メール本文はサーバーに保存されません（分析時のみ一時的に処理されます）
